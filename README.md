## Running locally

You will need `npm` and Hugo installed already.

- To run the site locally: `hugo server -D`
- To access [Sveltia CMS](https://sveltiacms.app/en/docs/intro) locally go to [/admin](http://localhost:1313/admin/)
- To run [accessibility checker](https://www.npmjs.com/package/accessibility-checker): `npm run test:a11y`
